# My vim configuration

## Usage

```
mkdir ~/.vim
cd ~/.vim
git clone git@github.com:nhatnm0612/my-vimrc.git .
cd pack/plugins/start/vim-surround
git clone https://github.com/tpope/vim-surround .
cd ../nerdtree
git clone https://github.com/scrooloose/nerdtree .
```

## Start

```
cd ~/.vim
vi vimrc
:source %
```
